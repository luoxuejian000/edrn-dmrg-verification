**协议状态：** 公开征求意见。本协议受元反思律约束，任何人在实验开始前均可通过GitHub Issues提出修改建议。在首次DMRG计算正式启动前，所有条款保持可修正。

## 1. 方法论准则

本实验严格遵循以下原则。它们不是装饰性的哲学声明，而是对每一个实验设计决策具有约束力的操作规范。

**实践介入论：** 实验本身是对理论场域的主动介入。我们选择的工具（DMRG）和定义的拓扑集合，构成了对真实物理场域的特定切割。任何工具都在放大某些信号的同时遮蔽另一些信号。我们必须诚实地审视这些切割的边界——明确哪些物理效应被当前设计所捕捉，哪些可能被系统性地遗漏。

**严防工具理性悖论：** 任何实验设计都可能存在隐藏的混淆变量。本协议的核心目标是在数据产生之前，通过公开的压力测试来识别并控制这些混淆变量。坚决杜绝在数据产出后，通过选择性报告、参数微调或重新定义"成功标准"来让数据"对齐"理论。所有U值和L值使用统一的DMRG参数。若某数据点无法收敛，记录哨兵值-999并标注原因，不强行修复。

**旗帜鲜明地反对"对齐"：** 本实验不预设任何预言一定成立。所有数据将并排放置，原始呈现。无论结果显示正相关、无关系，还是出现意外，都将被诚实地记录和报告。合作者的署名权和学术利益独立于实验结果的方向——正面发现与负面结果具有同等发表价值。这一对称性是防止"对齐"动机的制度保障。

**只看病，不开方：** 本实验仅诊断物理系统的真实状态。不对"这意味着什么"做提前判断，不提供任何规范性结论。判断权完全留给研究者共同体。

## 2. 预言一：A(C)关系多拓扑检验

### 2.1 科学问题与历史教训

**原始猜想：** 自旋能隙的有限尺寸标度形式 Δ_s = A(C)/L 中的前置因子A(C)，可能与网络的拓扑连通指数C有关。这一猜想源于"关系先于实体"的核心公设——如果系统的全部可观测量性质由关系网络决定，那么自旋激发的能隙前置因子也应反映网络的拓扑特征。

**第一次实验的教训：** 在单变量变化（改变链长L）的线性链上，C是L的确定性函数。因此，通过改变L来观察A与C的关系，是工具理性悖论的典型案例。A和C都随L平滑漂移，高R²是统计上的必然，而非物理上的发现。这一教训的直接推论是：任何对A(C)关系的有效检验，必须在固定L的条件下进行，且自变量C必须独立于链长L而变化。

**本实验目标：** 在固定链长L的条件下，通过改变晶格拓扑结构来产生不同的C值，独立检验A是否追踪C。如果A在不同拓扑、相同L下坍缩到同一条A(C)曲线上，则猜想获得正面支持。如果A与C无单调关系，则记录为负面结果。两种结果同等重要，同等值得发表。

### 2.2 实验设计：固定L，多拓扑扫描

**固定链长：** 选择两个链长，以检验结果的L收敛性：L = 40 和 L = 60。两个链长各自独立分析，互不对齐。若L=40和L=60的数据呈现不同趋势，则判定存在有限尺寸瞬态效应，该趋势不能被解读为体态规律。

**候选拓扑及门控条件：** 候选拓扑必须通过以下"门控条件"，才有资格进入A(C)分析。门控条件的存在本身，就是对工具理性悖论的一次预先防范——它防止了将不满足基本物理前提的拓扑错误地纳入分析。

门控条件：自旋扇区必须是无能隙的。对于每个(L, 拓扑)，首先计算自旋能隙Δ_s。如果Δ_s × L随L增大而趋于常数（即Δ_s ~ 1/L），则判定为无能隙，通过。如果Δ_s趋于非零常数（即存在自旋能隙），则判定为有能隙，该拓扑被排除。这不是"坏数据"，而是类别错误——该拓扑不属于本实验的检验范围。将有能隙拓扑的A值与无能隙拓扑的A值混入同一拟合，是用一个类别的数据去回答另一个类别的问题。

基于此门控，候选拓扑如下：

| 拓扑类型 | 描述 | 自旋能隙状态 | 是否通过门控 |
|:---|:---|:---|:---|
| 均匀开链 | 标准一维链，开边界 | 无能隙（已知） | ✅ 基准数据 |
| 均匀周期链 | 标准一维链，周期边界 | 无能隙（已知） | ✅ 需验证 |
| 单键缺陷链 | 均匀开链中，改变某一特定键的跃迁强度，从0.5t到1.5t连续变化 | 无能隙（已知，体能隙保持为零） | ✅ 连续调节旋钮 |
| ~~两腿梯子~~ | 耦合的两条链 | 有自旋能隙 | ❌ 被排除 |
| ~~二聚化链~~ | 交替键强弱变化 | 任意非零二聚化均打开自旋-Peierls能隙 | ❌ 被排除 |

单键缺陷作为"连续调节旋钮"，是产生C变异的主要来源。开链和周期链仅提供离散锚点。DanceNitra指出，单键缺陷是一个边界扰动，在重整化群下会流向有效切断的链。其C变化可能部分地是一个有限尺寸瞬态。因此，必须在两个L值下都进行检验，确认A(C)趋势L收敛后，才能将其解读为体态关系。

**控制变量：**
- 模型：一维单带Fermi-Hubbard模型，U/t = 4，半满填充。所有拓扑统一使用这些参数。
- DMRG参数：所有计算统一使用χ_max=100、svd_min=1e-10、mixer开启。若某拓扑在此参数下不收敛，记录哨兵值，不单独调参。
- 配位数：除周期链外，所有拓扑保持配位数z=2。
- 可观测量：电荷能隙ΔE = E(N) - E(N-1)，自旋能隙Δ_s = E(S=1) - E(S=0)，拓扑连通指数C，前置因子A = Δ_s × L。

### 2.3 数据收集、分析与判定标准

**数据收集与记录：** 所有原始数据以CSV格式保存。每行包含：拓扑类型、L值、缺陷强度（如适用）、E(N-1)、E(N)、电荷能隙、自旋能隙、C、A、计算耗时、时间戳。

**分析步骤：**
1. 门控筛选：排除所有不满足Δ_s ~ 1/L的拓扑数据点。将通过门控的标注为"有效数据点"，未通过的标注为"类别错误-排除"。
2. 散点图可视化：将通过门控的(C, A)数据点绘制在散点图上。不同拓扑类型使用不同标记，两个L值使用不同颜色。散点图是呈现原始数据的最诚实方式。
3. 结果判定：
    - 正面结果：若A在不同拓扑、相同L下沿单一A(C)曲线坍缩，且L=40和L=60的曲线趋势一致，则猜想获得正面支持。
    - 负面结果：若A与C无单调关系，或仅在某特定拓扑类型内变化，记录为负面结果。
    - 不确定结果：若L=40和L=60呈现不同趋势，则判定存在有限尺寸瞬态效应，需诚实记录该不确定性。

**禁止事项：** 不对数据进行加权处理。不剔除任何离群值（除非已被门控条件正式排除）。不尝试多种非线性拟合来寻找"最佳"函数形式。如果通过门控的有效数据点少于4个，或C值的变异范围小于其均值的一定比例，不进行任何回归分析，只呈现散点图并做出定性陈述。

## 3. 预言二：L扫描检验方案（有限尺寸效应判决性实验）

### 3.1 科学问题

固定U=1.0，扫描L=120, 160，结合已有的L=20, 40, 60, 80数据，检验电荷标度指数是否随L增大而向零漂移。这是对DanceNitra提出的有限尺寸效应质疑的直接检验。DanceNitra指出，一维半满Hubbard模型在任意U>0时都是Mott绝缘体，唯一的临界点在U=0。若电荷标度指数在固定U下随L增大而向零漂移，则证明预言二的原始表述（电荷与自旋的标度行为在有效临界U值处发生交换）是有限尺寸假象。

### 3.2 数据来源

- 脚本：`L_scan_U1.py`
- 输出文件：`data/L_scan_U1.csv`
- 已有数据：U=1.0, L=20, 40, 60, 80（来自预言二的U扫描）
- 补跑数据：U=1.0, L=120, 160（本实验产出）

### 3.3 判定标准

- 若α_charge随L增大单调向零漂移，支持有限尺寸效应解释。预言二的原始表述（标度交换）被否定。
- 若α_charge在更大L下保持稳定（不随L增大而向零漂移），预言二获得更强支持。
- 无论哪种结果，自旋标度指数α_spin应始终保持约-1，作为内部一致性检验。

### 3.4 与预言三的关系

本实验产出的L扫描数据，同时用于检验预言三（α_charge + α_spin = -1）。在更大的L值下，有限尺寸效应减弱，α_charge + α_spin应更接近-1。L扫描因而是预言二和预言三的共同判决性实验。

## 4. 预言三：跨扇区标度指数守恒检验方案

### 4.1 科学问题

检验α_charge + α_spin = -1是否在所有U>0的热力学极限下成立。这一预言源于矛盾动力论和沉默失谐原理：当电荷扇区的临界涨落被Mott能隙"沉默"时，矛盾没有消失，而是完整地转移到了自旋扇区。两个扇区的标度指数之和保持守恒，恒等于-1。

### 4.2 数据来源

本实验不需要额外的DMRG计算。所有数据来自已有实验：
- U=0.5, 1.0, 2.0, 4.0的U扫描数据（脚本：`predict2_U_scan.py`，输出文件：`data/predict2_U_scan_v2.csv`）
- U=1.0的L扫描数据（脚本：`L_scan_U1.py`，输出文件：`data/L_scan_U1.csv`）

预言三是对已有数据的重新分析，而非新的数据收集。

### 4.3 判定标准

- 对每个U值，计算α_charge + α_spin。
- 若所有U值下的和均接近-1，且随L增大单调收敛于-1，则预言三获得支持。
- 若和在某个U值下显著偏离-1（偏差超过0.3），且无法用有限尺寸效应解释，则预言三被否定。
- 若和在U=0.5时因有限尺寸效应偏大，但在U=4.0时精确收敛于-1，且L扫描数据进一步确认收敛趋势，则判定为"有限尺寸效应下的收敛行为，与预言三一致"。

### 4.4 与预言二的关系

预言二和预言三共享L扫描数据。若L扫描证实α_charge漂向零，则同时检验了两个预言：预言二的原始表述被否定，预言三的α_charge + α_spin = -1获得更大L值下的进一步支持。两个预言的检验是协同的，而非独立的。

## 5. 协作方式、署名与数据管理

**独立DMRG交叉验证：** DanceNitra团队将使用其自身的DMRG实现，对相同拓扑和参数进行独立计算。独立交叉验证是防范单一方法系统偏差的最有效手段。如果两套独立数据得到相同的结论，该结论的可信度将显著提高。如果两套数据存在差异，差异本身就是重要的诊断信息——它可能揭示收敛问题或方法偏差。

**数据并排：** 所有结果将以"并排放置"的方式呈现在最终报告中，不进行平均或合并。各方数据独立可见，任何差异都将被保留和讨论。

**署名：** 无论实验结果如何，所有实质性贡献者共同署名。正面发现与负面结果具有同等发表价值。原始猜想由李广好提出。方法论纠正和实验方案设计由DanceNitra贡献。TAT跨框架验证由Marat Sultanov贡献。

## 6. 阳性与阴性对照：TAT跨框架验证

为在同一次协作中验证TAT诊断工具的跨领域有效性，特设以下对照组。该设计源于Marat Sultanov的原始TAT分析及其后续的自我修正。

**阴性对照（无相变系统）：** 扫描U/t ∈ [0.5, 4.0]的半满Hubbard链。DanceNitra已确认，此参数范围内不存在有限U的相变，唯一的临界点在U=0。因此，TAT在此扫描中应保持沉默。该阴性对照直接检验TAT是否会在无相变系统中错误地"发现"信号——即检验TAT是否继承了Marat最初发现的"诚实的沉默"品质。

**阳性对照（有相变系统）：** 备选系统三选一或全部：
- 二聚化扫描：在均匀链中引入交替跃迁δ，扫描δ ∈ [0, 0.5]，临界点位于δ=0。
- J1-J2链：扫描J2/J1 ∈ [0, 0.5]，无能隙-二聚化转变点位于0.2411 (Okamoto-Nomura)。
- XXZ链：扫描各向异性Δ ∈ [0, 2]，BKT转变点位于Δ=1。

如果TAT在阴性对照中保持沉默，在阳性对照中检测到已知相变，则TAT的跨领域有效性获得强有力的正面验证。如果TAT在阴性对照中错误触发，或在阳性对照中沉默，则记录为TAT的重要局限性。无论哪种结果，都是诚实的科学产出。

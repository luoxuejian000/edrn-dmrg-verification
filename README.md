# edrn-dmrg-verification

基于密度矩阵重整化群（DMRG）的电子离域关系网络（EDRN）标度行为独立验证项目。

## 项目背景

本仓库为独立研究者李广好的EDRN理论框架提供公开、透明、可复现的数值验证平台。

**核心科学问题：**

一维半满Hubbard模型在U/t = 4时，电荷压缩率χ随链长L的标度指数α是多少？

**理论预言：** α = 1.75（原始预言，电荷扇区）→ 经自我修正后，临界区定位于自旋扇区（Δ_s ~ 1/L，α = -1）

**最终论文：** 见本仓库 `一维电子离域关系网络的电荷与自旋响应：理论预言、数值检验与基于关系网络的边界修正.pdf`

## 当前计算结果（两扇区法，U/t = 4）

### 电荷压缩率

| L | 电荷能隙 ΔE | 压缩率 χ = 1/ΔE |
|:---|:---|:---|
| 20 | 1.246324 | 0.802360 |
| 40 | 1.313976 | 0.761049 |
| 60 | 1.331761 | 0.750885 |
| 80 | 1.338900 | 0.746882 |
| 100 | 1.342315 | 0.744982 |
| 120 | 1.344094 | 0.743996 |
| 160 | 1.345489 | 0.743224 |
| 200 | 1.346283 | 0.742786 |

拟合结果：α ≈ -0.03，R² = 0.94

结论：电荷压缩率不显示幂律标度。系统处于Mott绝缘相，电荷能隙 ≈ 1.35t。

### 自旋能隙

| L | Δ_s | Δ_s × L |
|:---|:---|:---|
| 20 | 0.144979 | 2.900 |
| 40 | 0.076668 | 3.067 |
| 60 | 0.052320 | 3.139 |
| 80 | 0.039769 | 3.182 |
| 100 | 0.032086 | 3.209 |

拟合结果：α = -1.05 ± 0.03，R² = 0.996

结论：自旋能隙按~1/L衰减至零。自旋扇区处于无能隙临界相。

## 如何参与协作验证

任何研究者均可参与本项目的独立验证。验证结果以并排放置（side-by-side）的方式呈现，不做统一标准化或加权合并。每位参与者的数据保留原始形式和命名。

### 方法一：使用本仓库提供的参考脚本

```bash
# 安装依赖
pip install tenpy

# 运行电荷压缩率计算（修改L值后）
python run_compress.py

# 运行自旋能隙计算（修改L值后）
python run_spin_gap.py
```

技术细节

· DMRG库： TenPy 1.1.0（纯Python模式，无Cython编译）
· Python版本： 3.11.9
· 模型： 一维单带Fermi-Hubbard链，开边界条件
· 参数： U/t = 4，半满填充
· 最大键维度： χ_max = 100
· 计算方法： 两扇区法（基于粒子-空穴对称性，避免N=L+1收敛困难）

所有计算均在本地计算机上完成，未使用任何云端或远程服务器。完整审计日志见各脚本输出。

**协议状态：** 公开征求意见，受元反思律约束，任何人在实验开始前均可提出修改建议。

## 1. 方法论准则

本协作严格遵循晶脉哲学的核心原则：

- **实践介入论：** 实验本身是对理论场域的主动介入。我们选择的工具（DMRG）和定义的拓扑/参数，构成了对真实物理场域的特定切割。必须诚实地审视这些切割的边界。
- **严防工具理性悖论：** 任何实验设计都可能存在隐藏的混淆变量。本协议的目标是在数据产生之前，通过公开的压力测试来识别并控制这些混淆变量，而非在数据产出后用选择性报告或参数微调来“对齐”理论。
- **旗帜鲜明地反对“对齐”：** 本实验不预设任何预言一定成立。所有数据将并排放置，原始呈现。无论结果显示正相关、无关系，还是出现意外，都将被诚实地记录和报告。合作者利益独立于实验结果方向。
- **只看病，不开方：** 本实验仅诊断物理系统的真实状态。不对“这意味着什么”做提前判断。判断权留给研究者共同体。

---

## 2. 科学背景与后续探索方向

在先前的协作中，我们完成了对EDRN框架三个核心预言的系统检验。预言二（标度交换）被否定，预言三（跨扇区标度恒等式）被确认为自旋-电荷分离的数值体现。预言一（自旋能隙前置因子与拓扑指数的关系）在初次检验中因变量混淆而被修正，其物理根源被重新定位为自旋关联穿透深度随局部键强度的变化，即Kane-Fisher弱链图景在Mott自旋扇区中的确认。

基于以上基础，本协议旨在组织下一阶段的探索，重点检验以下标准理论未明确预言或EDRN框架可独特推导的物理效应：

1. 周期边界对自旋激发前置因子的影响（检验“关系先于实体”）
2. 弱耦合区自旋临界性的普适性
3. 局域拓扑异质结对自旋激发的调控
4. 跨扇区标度恒等式在多系统中的普适性
5. 非平衡驱动下的跨扇区矛盾守恒

所有实验将继续遵循统一的DMRG参数和开源协作模式。

## 3. 实验一：周期链与开链的自旋能隙前置因子对比

**科学问题：** 在所有局域跃迁参数完全相同的条件下，仅改变边界条件（开边界→周期边界），自旋能隙前置因子 A = Δ_s × L 是否发生显著变化？

**实验设计：**
- 模型：一维单带Fermi-Hubbard模型，U/t=4，半满填充
- 几何：L=40 开链与周期链
- 所有键强度保持 1.0t
- 观测量：自旋能隙 Δ_s、前置因子 A、拓扑指数 C、电荷能隙 ΔE（门控条件）
- DMRG参数：χ_max=100, svd_min=1e-10, mixer启用

**门控条件：** 电荷能隙 ΔE > 0 必须成立，以确认系统处于Mott绝缘相。

**当前状态：** 自旋能隙数据已获取（A周期≈3.526, A开链≈3.067，差异约15%）。电荷能隙计算在周期边界下存在收敛困难，需要独立DMRG验证或算法优化。

**待协作：** Drahos独立代码验证周期链电荷能隙。

## 4. 实验二：弱耦合区自旋临界性检验

**科学问题：** 在弱耦合区（U/t=0.5），自旋扇区是否仍然保持海森堡普适类的临界标度行为？

**实验设计：**
- 模型：一维单带Fermi-Hubbard模型，U/t=0.5，半满填充
- 几何：L=40 开链
- 观测量：自旋能隙 Δ_s、乘积 A = Δ_s × L
- 对比标准：Drahos用Bethe ansatz计算的精确自旋波速 πv_s(U=0.5)

**当前状态：** L=40 数据已获取（A≈5.497），与理论预期一致。可补充 L=60 进行有限尺寸标度验证。

## 5. 实验三：拓扑异质结对自旋激发的调控

**科学问题：** 在同一条链上引入两个不同强度的局部键，能否在穿透深度修正后仍创造出两个具有不同自旋激发前置因子的区域？

**实验设计：**
- 模型：一维单带Fermi-Hubbard模型，U/t=4，半满填充
- 几何：L=60 开链
- 在 L/4 处设置 t=1.5t，在 3L/4 处设置 t=0.5t
- 观测量：全局自旋能隙 Δ_s、左右两侧的局部穿透深度 ξ_left, ξ_right、有效前置因子 A_left, A_right
- 注意：需要先用穿透深度修正消除纯粹由键强度差异引起的效应，再检验是否存在剩余差异。

**当前状态：** 实验设计阶段。需讨论穿透深度修正后如何定义局域前置因子以避免循环论证。

## 6. 实验四：跨扇区标度恒等式的跨系统验证

**科学问题：** α_charge + α_spin = -1 是否在其他强关联系统中也成立？

**实验设计：**
- 无需运行新DMRG计算（现阶段）
- 利用公开可用的DMRG数据集或量子蒙特卡洛数据，提取电荷和自旋能隙的标度指数
- 检验等式在多种模型和维度下的普适性

**当前状态：** 一维Hubbard模型跨参数（U/t=0.5至4.0）验证已完成。公开数据集搜寻中。

## 7. 实验五：非平衡驱动下的跨扇区矛盾守恒

**科学问题：** 在周期性改变局部键强度的驱动下，静态标度指数关系 α_charge + α_spin = -1 是否仍然在每个瞬时成立？

**实验设计：**
- 模型：一维单带Fermi-Hubbard模型，U/t=4，半满填充
- 几何：L=40 开链
- 驱动：中心键强度在 0.5t 与 1.5t 之间取多个离散快照（如五个）
- 每个快照独立运行静态DMRG，计算自旋能隙和电荷能隙
- 分析等式是否在驱动序列中保持稳定

**当前状态：** 脚本已就绪，待计算资源空闲时运行。
